# CODEX PHASE 4 – Qualität, SEO, Mobile und Launch

## Ziel

Die Website wird produktionsreif gemacht.

## Aufgaben

### Design-QA
- Desktop 1448 × 1086 gegen Referenzen prüfen
- Mobile 390, 430, 768 prüfen
- Header, Hero, Karten, Footer prüfen
- Startseitenhintergrund final korrigieren
- keine horizontalen Scrollbars

### Inhalt-QA
- alle Texte Deutsch
- keine Lorem-Ipsum-Texte
- keine falschen Eventdaten
- keine fremden Fighter
- nur 4 echte Champions
- Ticketpreise prüfen
- Social Links prüfen
- E-Mail prüfen

### Technik-QA
- `pnpm lint`
- `pnpm build`
- keine Console Errors
- keine fehlenden Bilder
- keine toten Links
- keine TypeScript-Fehler

### SEO
- Seitentitel
- Meta Descriptions
- OpenGraph
- Twitter Cards
- Sitemap
- Robots
- 404-Seite

### Rechtliches
- Impressum
- Datenschutz
- Cookie-Hinweis nur falls nötig
- Kontaktformular-Datenschutztext

### Performance
- Bilder komprimieren
- Hero Images optimieren
- Lazy Loading prüfen
- Fonts stabilisieren
- `latest` Dependencies pinnen

## Fertig, wenn

- Website live stabil ist
- Mobile sauber ist
- SEO-Grundlage vorhanden ist
- keine offensichtlichen Designfehler sichtbar sind
- alle Links funktionieren
