# Premium UI Polish

Feinschliff heisst hier: die Umsetzung muss wie ein professionelles Kampfsport-Poster wirken, nicht wie eine generische Website mit dunklem Hintergrund.

## Pruefpunkte

- Hero: massive Headline, klare CTA-Zone, Bild mit Vignette, roter/goldener Akzent, keine Text-Card im Hero.
- Header: horizontal, sticky, kompakt; roter aktiver Unterstrich; Tickets als staerkster CTA.
- Sections: harte horizontale Module, goldene/rote Trennlinien, kontrollierte Dichte statt leerer Flaeche.
- Cards: kantig, duenne rote/goldene Borders, Grunge/Scratch/Diagonal-Linien, keine glatten Standard-Boxen.
- Buttons: Primary rot, Secondary Outline, Premium Gold/Beige; klare Hover-/Focus-Zustaende.
- Typografie: uppercase Poster-Hierarchie; keine viewport-abhaengige Schriftgroesse; keine negative Letter-Spacing-Experimente.
- Lesbarkeit: Text liegt ueber Overlay, nie direkt auf unruhigem Bild.
- Empty States: ehrlich "wird bald bekanntgegeben" oder "wird nachgetragen", nie Lorem Ipsum.
- Fehlerzustaende: sichtbar, deutsch, markenkonform; keine nackten Browser-Alerts als finale UI.

## Vorgehen

1. Vergleichbare bestehende Komponenten lesen, damit Buttons/Cards nicht auseinanderlaufen.
2. Kleine, gezielte Korrekturen bevorzugen: Spacing, Border, Overlay, Typografie, Bildcrop.
3. Nach UI-Aenderungen Mobile und Desktop gegenpruefen.
4. Bei Designkonflikt gewinnt `smashtime_claude_design_brief_final.md`.
